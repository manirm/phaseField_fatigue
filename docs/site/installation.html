Installation Guide

Setting up a scientific computing environment for FEniCSx and PETSc requires specific tools. We recommend using Conda (Mamba/Miniforge) for the most stable experience.

1. Cross-Platform Quick Start

conda create -n fenicsx -c conda-forge fenics-dolfinx gmsh wxpython mpich
conda activate fenicsx

2. OS-Specific Details

macOS (Intel & Apple Silicon)

  • Silicon Chips: Use Miniforge for native ARM64 performance.
  • Homebrew Conflict: Ensure you run brew unlink mpich if you have a system-wide MPI installed.

Windows (via WSL2)

Direct Windows execution is not supported for high-performance physics. You must use WSL2 (Windows Subsystem for Linux).

  1. Install Ubuntu 22.04 from the Microsoft Store.
  2. Inside the WSL terminal, follow the Conda Quick Start above.
  3. Ensure WSLg is up to date to support native graphical windows.

Linux (Ubuntu)

You can use the native PPA for the highest performance:

sudo add-apt-repository ppa:fenics-packages/fenics
sudo apt update
sudo apt install fenicsx gmsh
MPI Warning If the application crashes on launch with "MPI Error", ensure you are running it within the active Conda environment where mpich was installed.
