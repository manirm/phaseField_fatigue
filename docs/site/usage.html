Application Walkthrough

This guide explains the function of every component in the Phase-Field Fatigue Suite GUI to help you set up and analyze your scientific runs.

1. Specimen Configuration

The **Control Panel** (left pane) is where you define the geometry and material properties.

  • Template Choice: Select between CT (Compact Tension), SENB (Single Edge Notch Bending), or CCT geometries.
  • Preview Mesh: Regenerates the Gmsh topology based on your selected length scale \(\ell\) and specimen dimensions. Always preview before running to ensure refinement is sufficient.

2. Material Parameters

You can select from the **Material Presets** (e.g., Al7075, AISI 4340) or define custom values:

  • Young's Modulus (E): Elastic stiffness.
  • Fracture Toughness (Gc): Energy release rate for crack initiation.
  • Length Scale (L): Controls the width of the regularized crack profile. Smarter, narrower cracks require finer meshes.

3. Multi-Physics Dashboard

The **Settings** tab (within the main Notebook) allows you to enable environmental effects:

  • Hydrogen Transport: Enables the stress-assisted diffusion solver.
  • HEDE Degradation: Controls the decohesion sensitivity \(\chi\).
  • D Coefficient: Effective diffusion speed through the lattice.

4. Running and Monitoring

Once parameters are set, use the bottom-right buttons:

  • Mode - Quick: Rapid life estimation using analytical J-integration.
  • Mode - Full: Research-grade 3D variational simulation.
  • Stop: Gracefully interrupts the MPI process and saves current XDMF checkpoints.
Tip: Post-Processing All simulation results are saved in the results/ directory. Use Paraview to visualize the .xdmf files for 3D damage contour mapping.
