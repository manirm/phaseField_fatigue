The Phase-Field Fatigue Suite is a professional 3D fracture mechanics application designed for high-fidelity crack propagation analysis. It integrates advanced multi-physics models, including Hydrogen-Assisted Fracture and stress-assisted diffusion.

Installation & Environment

To ensure numerical stability and performance, we recommend using Conda or Mamba to manage the scientific stack (FEniCSx, PETSc, and Gmsh).

Windows (via WSL2)

Native Windows is not supported for the solver engine. Please install WSL2 (Ubuntu 22.04+) and follow the instructions within the WSL terminal.

macOS (Intel & Apple Silicon)

conda create -n fenicsx -c conda-forge fenics-dolfinx gmsh wxpython mpi4py

Tip: Use Miniforge on M1/M2/M3 chips for native ARM64 performance.

Linux (Ubuntu 22.04+)

sudo add-apt-repository ppa:fenics-packages/fenics
sudo apt update
sudo apt install fenicsx gmsh

Quick Start

Follow these steps to generate your first 3D fracture simulation:

  • Select Specimen: Choose CT, SENB, or CCT from the dropdown.
  • Preview Mesh: Click Preview Mesh to verify refinement density.
  • Parameters: Set Young's Modulus (E) and Fracture Toughness (Gc).
  • Run: Select "Full Mode" for variational staggered analysis and click Run Simulation.

Batch Processing & DOE

Perform parametric sweeps using the Batch Manager tab. This feature supports automated Grid Searches. Simply enable the parameters you wish to vary, provide a comma-separated list of values, and click Generate & Run Batch. The suite will automatically plot result trends once the queue finishes.
